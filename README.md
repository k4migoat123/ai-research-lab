## About This Repository

This is my working research lab for AI evaluation, prompt engineering, and LLM benchmarking.

I regularly work with ChatGPT, Claude, DeepSeek, Qwen, Gemini, and open-weight models — comparing outputs, designing structured prompts, and documenting findings in a format useful for AI evaluation teams.

This repository is where I turn that work into organized, reusable knowledge.
